# Aldora Kids Learn Pro

Premium multilingual kids learning web application built with HTML, CSS, and JavaScript.

## Features
- Multi-language Learning
- Interactive Visual Cards
- Audio Pronunciation
- Mini Quiz System
- Reward & Star Progress
- Dark / Light Mode
- Responsive Design
- Search & Filter System
- Offline Ready
- Modern Enterprise UI

## Supported Languages
- Indonesian
- English
- German
- Japanese
- Korean
- Chinese

## Technology
- HTML5
- CSS3
- JavaScript
- Speech Synthesis API
- Local Storage
